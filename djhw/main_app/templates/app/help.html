{% blocktrans %}Sets Stats Table (SST){% endblocktrans %}

{% blocktrans %}Main elements{% endblocktrans %}
{% blocktrans %}

The SST is based on 3-sets tennis matches, data from 5-sets matches are excluded from a bias. Top half of the table shows data for players to win whole match, the bottom half shows stats for a next set. There are Filters Buttons under the table. You can use them to narrow bias on one of the following criterias:

ODD - Odds

The last Non-Inplay odds for a player represent its strength by combining a lot of factors. This filter allows you to see past player results from matches with same strength opponents.

SFC - Surface

This filter allows you to see past player results from matches on the same surface.

CNT - Country

This filter allows you to see past player results from matches on the same country. This factor can be important if a match takes place in native country of one players.

RND - Round

This filter allows you to see past player results from matches on the same tour stage. There are four stages:

  • Pre-Start - qualification rounds;
  • First stage - 1st round only;
  • Middle stage - from 2nd round till semifinal;
  • Last stage - semifinal and final

Please always look at the total count of matches. The bigger, the better. If this number is less then 20 then it is mostly not representative.

{% endblocktrans %}

{% blocktrans %}Sets Progress Selector (SPS){% endblocktrans %}
{% blocktrans %}To model the match progress from set to set you should use the Sets Progress Selector.

The Match Filters under the SST works for current modelled situation like before start of the match. SPS works like additional filter, so it is possible to get non-representative numbers for one or both teams. There are two posiible situations with score 1-1: from 1-0 (team 1 first advanced) or from 0-1 (team 2 first advanced). These are totally different situations with different stats.
