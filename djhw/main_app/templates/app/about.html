{% blocktrans %}What is already realized?{% endblocktrans %}

{% blocktrans %} Tennis Data

Now Sposta works to collect tennis data. There are four sources of data:

  • Historical results of tennis matches for last 3 years;
  • Average and max odds from Oddsportal.com;
  • Non-Inplay and Inplay odds from Betfair;
  • Livescores data.

The odds data is combined with livescores and presented in graph view. The players statistic from historical results is shown in table view. There are fresh matches are shown only but collected info stores in our offline Lab database.

Users

You can Sign-In and set your timezone and language in the user settings.

Languages

There are English and Russian languages available via User settings. Players names are not translated and I am not sure I'll do translate them.{% endblocktrans %}

{% blocktrans %}Future plans.{% endblocktrans %}

{% blocktrans %}

Wagers and Trades (WT)

Any User can make own suggestions about upcoming matches. Those suggestions can be just comments or WT. When match results will be available, WT grades and current User profit/loss calculates.

Users Statistics

Any User with WT has own statistic with profit/loss calculation, ROI and so on. Those Users are shown in Results Comparison Chart.

Paid services

User can mark any of his ot her sugestions as Premium, so only Premium Users can see them before match completes.

Subscription

Any User (Customer) can subcribe on suggestions from another User (Provider). To get Premium content Customer should buy a Subscription Plan from a Provider. Suggestions can be delivered via site, email and Telegram service.

Guarantees

Provider can give any guarantees on his suggestions. If the contract is violated, Sposta is a third party in the dispute. Customer can drop contract at any time. In this case the rest of the money must be returned to Customer. The mechanism for the return of funds must be carefully worked out.

Other Sports

Autumn, 2017 I plan to start collect Soccer and Basketball data.
